Support FAQ

Common technical questions and setup issues resolved.

To avoid hitting Discord API rate limits, periodic leaderboard nickname updates have been disabled. User nickname rank prefixes now update immediately after points are awarded at the end of matches. If point additions still don't update nicknames, make sure the bot's highest role is positioned above the target user's role in your Discord Server Settings.

To add helper bots, create up to 5 individual Discord bots in the Discord Developer Portal, invite them all to your server, and set their credentials in your environment configurations (TOKEN_1 to TOKEN_5). The bot's Rank Manager will automatically log them in and distribute nickname updates across them.

Use the prefix commands !addperm @role permissionName or !removeperm @role permissionName to grant or revoke specific command authorization (such as cancel, mvp, blacklist, or coins) to custom roles.
